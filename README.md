# 🚀 Proxy Config Hub: VLESS + HAPP + V2RayTun

Добро пожаловать в репозиторий, где собраны готовые конфигурации и подписки для безопасного и обходного интернета. Здесь вы найдёте настройки для современных прокси-протоколов.

## 📦 Что внутри?

Этот репозиторий содержит **авто-обновляемые подписки (subscription links)** и примеры конфигураций для следующих технологий:

| Тип | Протокол | Назначение |
|------|----------|-------------|
| **VLESS** | VLESS + REALITY / WebSocket (gRPC) | Высокая скорость, маскировка под обычный HTTPS-трафик |
| **HAPP** | Hysteria / Hysteria2 (часто называют HAPP) | Обход DPI, работа даже при плохом соединении (UDP-based) |
| **V2RayTun** | Tun2Ray / V2Ray с TUN-режимом | Системная маршрутизация всего трафика (VPN-подобный режим) |

## 📄 Форматы файлов

- `.json` / `.yaml` — для ручного импорта в **V2Ray**, **Xray**, **Hiddify**, **Nekoray**.
- `.txt` / `.link` — прямые подписочные ссылки (subs) для **Sing-box**, **V2Box**, **Shadowrocket**, **Stash**.
- `.mobileconfig` — для iOS (если есть).
